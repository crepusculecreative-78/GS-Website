# GS-Website
# ⛺ Globalys Solutions – Chapiteaux Sur Mesure Plateforme web vitrine moderne et responsive pour **Globalys Solutions**, spécialisée dans la création, fabrication et installation de chapiteaux et structures événementielles personnalisées.
## ✨ Fonctionnalités
- **Design Premium** : Esthétique sombre et dorée avec animations au défilement.
- **Catalogue de Produits** : Présentation détaillée des différents types de structures (Bi-Pente, Stretch, Tipi, etc.).
- **Portfolio Dynamique** : Réalisations filtrables par catégorie d'événement (Mariages, Entreprise, Festivals).
- **Formulaire de Devis Intelligent** : 
  - Système hybride : `MailHog` pour les tests locaux.
  - Integration `Netlify Forms` pour la réception des demandes en production.
- **Performance & SEO** : Structure sémantique optimisée et chargement rapide.
## 🚀 Technologies utilisées
- **Frontend** : HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Backend (Développement)** : Node.js & Express (pour le serveur d'envoi d'emails local).
- **Déploiement** : Prêt pour Netlify ou Vercel.
## 🛠️ Installation locale
1. Clonez le dépôt.
2. Installez les dépendances : `npm install`
3. Lancez le serveur de test : `node server.js`
4. Ouvrez `index.html` dans votre navigateur.
## 🌐 Déploiement
Le projet est configuré pour un déploiement instantané sur **Netlify**. Les formulaires de contact sont gérés automatiquement sans configuration supplémentaire nécessaire une fois en ligne.
